Redis数据类型

  • key：String类型，最大512MB
  • value：
    • Binary-safe strings（字符串/整数）
    • Lists 按照插入顺序保存的字符串集合
    • Sets 元素唯一 未排序的集合
    • Sorted sets 每个元素关联一个浮点数（score），元素根据这个score排序
    • Hashes
    • Bit arrays
    • HyperLogLogs
    • Streams
